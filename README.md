🐍 Curso Básico de Python

Este repositório contém todos os códigos e exercícios desenvolvidos durante o meu curso básico de Python. 
O foco principal deste projeto é documentar o aprendizado através da aplicação prática rigorosa, 
escrevendo os scripts e testando a lógica em tempo real enquanto acompanho o conteúdo das aulas.
🎯 Objetivo

Registrar a evolução do aprendizado em programação, construindo uma base sólida na sintaxe do Python, 
e criar um portfólio inicial que sirva de material de consulta rápida para estruturas fundamentais.
